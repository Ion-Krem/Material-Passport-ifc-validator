IFC

IFC Material Passport Validator

Version 1.0.0

Standalone validation tool for IFC building information models. Checks completeness of classification codes, base quantities, materials, and property sets required for material passport analysis and whole life carbon reporting.

Created by
Ioannis Kremmydas
Built with
Python, IfcOpenShell, Chart.js
Reference
Madaster Documentation

How to Use

  1. Start a validation. Click "↩ New Validation" or launch the app.
  2. Pick your files. Select a folder containing IFC files, or pick individual files.
  3. Wait for processing. The progress bar shows each file as it completes.
  4. Review the Dashboard. Summary cards, loaded models, and charts give an overview of all models.
  5. Open the Details tab. See per-element issues with sortable, filterable tables.
  6. Export. Save as Excel for spreadsheet review, or PDF to send to trades.
